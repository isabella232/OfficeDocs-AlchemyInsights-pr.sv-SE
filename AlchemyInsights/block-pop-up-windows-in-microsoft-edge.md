# <a name="block-pop-up-windows-in-microsoft-edge"></a>Blockera popup-fönster i Microsoft Edge

1. Gå till **... (Inställningar och mer)**  >  **Inställningar**  >  **Webbplatsbehörigheter**.
2. Välj **Popup-fönster och omdirigerar**.
3. Aktivera **Blockera**.
