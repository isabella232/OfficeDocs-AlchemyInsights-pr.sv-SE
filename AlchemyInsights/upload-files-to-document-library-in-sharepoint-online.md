# <a name="upload-files-to-onedrive-or-sharepoint"></a>Ladda upp filer till OneDrive eller SharePoint

Du kan ladda upp filer till ett dokument bibliotek i SharePoint Online genom att helt enkelt dra dem från datorn och släppa dem i dokument biblioteket. Med OneDrive kan du synkronisera filer mellan datorn och molnet så att du kan komma åt dina filer var du än är. Du kan också arbeta med synkroniserade filer direkt i Utforskaren och komma åt dina filer även när du är offline. Mer information om hur du laddar upp och synkroniserar finns på länkarna nedan.

- [Ladda upp en mapp eller filer till ett bibliotek](https://support.office.com/article/upload-a-folder-or-files-to-a-document-library-eb18fcba-c953-4d45-8d90-8da66edeacdb)

- [Ladda upp med synkronisering](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-in-windows-615391c4-2bd3-4aae-a42a-858262e42a49)

- [Fil begränsningar (ogiltiga tecken, fil överföringens storlek, Sök vägs längder etc.)](https://support.office.com/article/invalid-file-names-and-file-types-in-onedrive-onedrive-for-business-and-sharepoint-64883a5d-228e-48f5-b3d2-eb39e07630fa)

- [Åtgärda OneDrive-synkroniseringsproblem](https://support.office.com/article/Fix-OneDrive-sync-problems-83ab0d8a-8400-45b0-8dcf-dc8aa8a6bcf8)

- [Åtgärda problem vid uppladdning av filer på OneDrive-webbplatsen](https://support.office.com/article/Fix-problems-uploading-files-on-the-OneDrive-website-9afcc4a0-e344-4bc9-9c9d-59d3e802247e)
