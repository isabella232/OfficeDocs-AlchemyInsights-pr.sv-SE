# <a name="integrate-a-third-party-cloud-app-with-azure-ad"></a>Integrate a third-party cloud app with ‎Azure AD

Förenkla åtkomsten och skydda dina molnprogram från tredje part, t.ex. Salesforce och Dropbox. Use the [Integrate a third-party cloud app with ‎Azure AD‎](https://go.microsoft.com/fwlink/?linkid=2157464) wizard to add apps and configure single sign-on, which reduces or eliminates sign-in prompts.
