# <a name="verify-your-domain-with-bluehost"></a>Verifiera din domän med Bluehost

Om du vill verifiera domänen måste du lägga till en TXT-post i domänregistratorn för att verifiera att du äger domänen. 

Följ först guiden Lägg [till domän genom att](https://admin.microsoft.com/Adminportal#/Domains) välja **Inställningar** \> **Domäner** och sedan Lägg **till domän.**
  
Stegvisa anvisningar finns i Create DNS records at Bluehost for Microsoft 365 (skapa DNS-poster på Bluehost för [Microsoft 365).](https://docs.microsoft.com/microsoft-365/admin/dns/create-dns-records-at-bluehost)
