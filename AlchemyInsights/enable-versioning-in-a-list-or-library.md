# <a name="enable-versioning-for-a-list-or-library"></a>Aktivera versionshantering för en lista eller ett bibliotek

1. Gå till listan eller biblioteket.
    
2. Klicka **Inställningar** och klicka sedan på **Inställningar för bibliotek** eller **Listinställningar**.
    
3. Klicka **på Versionsinställningar.**
    
4. Välj de inställningar du vill använda och klicka sedan på **OK.**
